# HM Review

## Purpose

Provide hiring-side critique that combines recruiter gate checks with hiring manager signal evaluation.

## Default Lens

Balanced dual-lens:
- Recruiter screen: clarity, scanability, keyword fit, level alignment.
- Hiring manager depth: impact, ownership, technical/leadership evidence.

## Use When

- Reviewing resumes, cover letters, or application answers.
- Stress-testing role alignment before submission.
- Identifying weak claims, missing evidence, or fuzzy ownership.

## Workflow

1. Extract target role, level, and expected scope.
2. Run recruiter lens and list pass/fail risks.
3. Run hiring-manager lens and list credibility gaps.
4. Provide prioritized fixes and a tightened rewrite.

## Required Output

- Verdict: Strong / Needs Revision / High Risk
- Recruiter lens findings (top 3)
- Hiring manager lens findings (top 3)
- Rewritten version (or strongest section rewrite)
- Final submission checklist
