# Tech Content Review

## Purpose

Improve technical writing while matching the right level of depth for the intended audience.

## Default Mode

Audience-adaptive:
- Infer audience from context.
- Balance clarity and rigor automatically.
- Keep technical accuracy intact.

## Use When

- Editing architecture write-ups or technical narratives.
- Preparing engineering-facing or executive-facing technical content.
- Translating deep technical ideas for mixed audiences.

## Workflow

1. Infer audience (executive, cross-functional, engineering).
2. Evaluate clarity, precision, and logical flow.
3. Adjust jargon density and depth.
4. Rewrite with clearer structure and preserved rigor.

## Required Output

- Inferred audience and style choice
- Accuracy or ambiguity flags
- Rewritten section(s)
- Optional alternate version for a second audience
